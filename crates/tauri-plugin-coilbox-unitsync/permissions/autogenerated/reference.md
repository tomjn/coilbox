## Default Permission

Allows the unitsync plugin's content-scan, minimap and thumbnail commands.

#### This default permission set includes the following:

- `allow-unitsync-scan`
- `allow-unitsync-minimap`
- `allow-unitsync-thumbnails`
- `allow-unitsync-game-info`
