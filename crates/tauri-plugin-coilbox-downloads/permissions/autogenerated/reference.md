## Default Permission

Allows the downloads plugin's commands (version, rapid browsing, springfiles/BAR listing, downloads).

#### This default permission set includes the following:

- `allow-dl-version`
- `allow-dl-repos`
- `allow-dl-versions`
- `allow-dl-download`
- `allow-dl-springfiles-list`
- `allow-dl-bar-maps`
- `allow-dl-download-map`
- `allow-dl-download-file`
