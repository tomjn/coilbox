## Default Permission

Allows the animation plugin's commands (COB disassembly, BOS->COB compilation, reveal-in-folder).

#### This default permission set includes the following:

- `allow-anim-cob-disasm`
- `allow-anim-bos2cob`
- `allow-anim-reveal`
