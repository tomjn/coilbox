## Default Permission

Allows the content plugin's commands (detect candidates, load/rescan state, scan/add/remove roots, list/verify engines).

#### This default permission set includes the following:

- `allow-content-candidates`
- `allow-content-state-load`
- `allow-content-rescan`
- `allow-content-scan-root`
- `allow-content-add-root`
- `allow-content-remove-root`
- `allow-content-list-engines`
- `allow-content-verify-engine`
- `allow-content-open-path`
